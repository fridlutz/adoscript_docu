CLOSE_ALL_NOTEBOOKS closes all notebooks of a certain model.

# Syntax:  

```adoscript
{% raw %}
CC "Modeling" CLOSE_ALL_NOTEBOOKS [ modelid:intValue ].


# --> RESULT ecode:intValue

{% endraw %}
```
{: .line-numbers}

# Parameters:  

|`modelid`|intValue|

# Returns:  

|`ecode`|intValue, Contains the error code or is 0 in case of success.|
