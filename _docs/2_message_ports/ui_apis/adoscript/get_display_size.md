Queries the size of the display.

# Syntax:  

```adoscript
{% raw %}
CC "AdoScript" GET_DISPLAY_SIZE	.

  # --> RESULT w:intValue h:intValue .
{% endraw %}
```
{: .line-numbers}

# Parameters:  



# Returns:  

|`w`|intValue, holds the width of the display in pixels.|
|`h`|intValue, holds the height of the display in pixels.|
