OUT appends a line to the text field of an output window.

# Syntax:  

```adoscript
{% raw %}
CC "AdoScript" OUT	winid:strValue text:strValue .

# --> RESULT ecode:intValue .
{% endraw %}
```
{: .line-numbers}

# Parameters:  

|`winid`|strValue, id of the window (string)|
|`text`|strValue|

# Returns:  

|`ecode`|intValue|
