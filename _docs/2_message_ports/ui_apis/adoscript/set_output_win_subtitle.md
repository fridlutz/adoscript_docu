SET_OUTPUT_WIN_SUBTITLE appends extra text to an output window's title.

# Syntax:  

```adoscript
{% raw %}
CC "AdoScript" SET_OUTPUT_WIN_SUBTITLE winid:strValue subtitle:strValue

#--> RESULT ecode:intValue .
{% endraw %}
```
{: .line-numbers}

# Parameters:  

|`winid`|strValue|
|`subtitle`|strValue|

# Returns:  

|`ecode`|intValue|
