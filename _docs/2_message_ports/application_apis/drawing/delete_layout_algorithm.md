CREATE_LAYOUT_ALGORITHM deletes a layout algorithm instance.

# Syntax:  

```adoscript
{% raw %}
CC "Drawing" DELETE_LAYOUT_ALGORITHM name:strValue

# --> RESULT ecode:intValue
{% endraw %}
```
{: .line-numbers}

# Parameters:  

none

# Returns:  

|`ecode`|intValue, Contains the error code or is 0 in case of success.|
