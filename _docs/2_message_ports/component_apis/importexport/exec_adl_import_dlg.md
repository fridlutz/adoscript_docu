EXEC_ADL_IMPORT_DLG starts the adl import dialog. This command has the same effect as if the user starts the import dialog in ADOxx.

# Syntax:  

```adoscript
{% raw %}
CC "ImportExport" EXEC_ADL_IMPORT_DLG
{% endraw %}
```
{: .line-numbers}

# Parameters:  

none

# Returns:  

none
